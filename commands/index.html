!seen <user> If user has said something in one of the bot's joined channel's it tells you what channel and when. Everyone

!shorten <url> Creates a bit.ly version of the url Moderator
!addmoderator <user> Add's user to the bots moderator list. DOES NOT make them a channel moderator. Moderator

!enter If a raffle is currently active, this will join the active raffle if the user has permissions. Everyone
!addAutoReply <keyword1>|<keyword2>|<keyword3>...|<reply> Sends a message to the channel when a message contains all of the keywords. Use %user% in the reply to send the users name in the reply. e.g. !addAutoReply hello|Hey there %user%! would return Hey there billy! When billy says "hello" in the chat. Moderator
!join When said in the bot's channel the bot joins your chat Everyone
!leave When said in YOUR channel makes the bot leave your channel Owner
!pcmrbot Gives a little bit of info about the bot Moderator
!raffle <participant level> Starts a raffle for the level of participant provided. Users must type !enter to enter the raffle. Moderator
!disableReplies Disables all confirmations made by the bot. e.g. telling a user the option they voted for. Moderator
!enableReplies Enables all confirmations made by the bot. e.g. telling a user the option they voted for. Moderator
!clear This clears the chat by running the /clear command in twitch. Moderator
!me Has the bot run the /me command, this is just a fun one like making the bot say "pcmrbot banned someone bad and has to be put in the corner." Moderator
!subscribers This toggles subscribers only chat. Owner
!gabe Posts links to official pcmasterrace channels and subsidiaries. Only run this if you are ok with advertising them. Moderator
!commercial [OPTIONAL time] Runs a commercial with an optional amount of time. Twitch will automatically attempt to fit a number of commercials to fit a time slot. Owner
!fight Makes the sender throw up his fisticuffs in a battle of honour. Everyone
!steamsales Posts a link to the steam database for steam sales. Moderator
!fight [target] The user of the command fights the target. Everyone
!punch [target] The user of the command punches the target. Everyone
!ko [target] The user of the command knocks out the target. Everyone
!fight [target] The user of the command realizes that they have killed the target and flees the scene. Everyone
